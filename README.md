Elementa — Open UI Component Library

---

What is Elementa?

Elementa is an open-source component library for buttons, cards, loaders, toggles, inputs, and background effects. Every component features a live interactive preview — no screenshots, no surprises. Copy the code, paste it into your project, done.

No npm install. No build step. No tracking. No cookie banners. Just clean, tested UI components that work.

> Built for: Developers, designers, agencies, and teams who need fast, reliable, and privacy-compliant UI components.

Features

- 40+ Components — Buttons, Cards, Loaders, Inputs, Toggles & Backgrounds
- Live Preview — Test every hover state, animation, and effect before copying
- Framework-Agnostic — React, Vue, Svelte, Tailwind, or pure HTML/CSS
- Zero Dependencies — Each component is standalone and lightweight
- WCAG 2.2 Tested — Accessibility badges show what's audit-safe
- EU-Hosted & GDPR-Compliant — No tracking, no Google Analytics, no third-party cookies
- MIT Licensed — Free for personal and commercial use
- Copy & Paste — No npm install needed. Just copy and use.
- CMD+K Search — Instant command palette to find any component
- Dark Mode Optimized — Every component looks great on dark backgrounds

Quick Start

Browse & Copy

1. Go to [ui.it-handwerk-stuttgart.de](https://ui.it-handwerk-stuttgart.de)
2. Search or browse categories
3. Click any component
4. Switch between Dark / Light / Grid / Transparent preview modes
5. Hit "Copy Code" — paste into your project

Install (Optional)

If you prefer installing over copying:


Example

Here's how simple it is to use an Elementa component:

Original (HTML/CSS):

```html
<button class="aurora-glow-btn">
  <span>Get Started</span>
</button>
```

```css
.aurora-glow-btn {
  position: relative;
  padding: 12px 32px;
  border-radius: 9999px;
  background: linear-gradient(135deg, #8b5cf6, #06b6d4);
  /* ... full CSS on the platform */
}
```

React version:

```tsx
import { AuroraGlowButton } from "@elementa/ui";

<AuroraGlowButton>Get Started</AuroraGlowButton>
```

Contributing

We welcome contributions! Here's how:

1. Create an account (free) at [ui.it-handwerk-stuttgart.de](https://ui.it-handwerk-stuttgart.de)
2. Build your component in HTML/CSS
3. Submit it via the upload form
4. We review — every component goes through quality checks
5. Go live — your component is published for thousands of developers

Requirements:
- Valid HTML5 & CSS3
- No external dependencies (or explicitly declared)
- Minimum 50 characters description
- Appropriate tags
- Respects `prefers-reduced-motion`
- WCAG 2.2 AA contrast minimum

Read the full [Contributing Guidelines](./CONTRIBUTING.md).

---

Quality Standards

Every submitted component goes through:

1. Automated code validation — HTML/CSS linting
2. WCAG 2.2 contrast check — Automated accessibility scan
3. Functional review — Manual testing by the team
4. Browser compatibility check — Chrome, Firefox, Safari, Edge

Review time: 24–48 hours. You'll be notified via email.
